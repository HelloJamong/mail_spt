Mail Support Extension 설정

Gemini API를 사용하여 메일 작성을 도와드립니다

API 키 설정

Gemini API 키가 필요합니다. 여기에서 무료로 발급받을 수 있습니다.

사용 방법

1

API 키 입력

위에서 Gemini API 키를 입력하고 저장하세요.

2

메일 작성 중 사용

메일을 작성하다가 텍스트를 선택한 후 확장 프로그램 아이콘을 클릭하세요.

3

모드 선택

• 모드 1: 맞춤법 및 문법 검사
• 모드 2: 메일 내용 수정 및 추천
• 모드 3: 한↔영 통역
