# 메일 서포트 (Mail Support)

Google Gemini API를 활용한 업무 메일 작성 도우미 Chrome Extension

## 소개

메일 서포트는 업무 메일 작성 시 문법 검사, 내용 개선, 번역 기능을 제공하여 보다 전문적이고 효과적인 커뮤니케이션을 돕는 Chrome 확장 프로그램입니다.

## 주요 기능

### 1. 맞춤법 및 문법 검사
- 작성한 메일 내용의 맞춤법과 문법을 자동으로 검사
- 오류가 있는 부분을 수정하고 개선된 텍스트 제공
- 수정 사항에 대한 간단한 설명 포함

### 2. 맞춤 메일 작성
- AI 기반 프롬프트를 통해 메일 내용을 더욱 전문적으로 개선
- 메일의 톤과 스타일을 비즈니스에 적합하게 조정
- 내용 구조화 및 표현 방식 개선 제안

### 3. 다국어 번역
- 한국어 ↔ 영어 자동 번역 기능
- 언어 자동 감지로 편리한 사용
- 해외 고객사와의 원활한 커뮤니케이션 지원

## 설치 및 설정

### 1. 확장 프로그램 설치

1. Chrome 브라우저에서 `chrome://extensions/` 접속
2. 우측 상단의 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 프로젝트 디렉토리 선택

### 2. API 키 설정

1. **Gemini API 키 발급**
   - [Google AI Studio](https://aistudio.google.com/app/apikey)에서 무료 API 키 발급
   - 계정 로그인 후 "Create API Key" 클릭

2. **API 키 등록**
   - 확장 프로그램 설정 페이지 열기
   - 발급받은 API 키 입력 및 저장
   - "API 연결 테스트" 버튼으로 정상 작동 확인

## 사용 방법

1. **텍스트 선택**
   - 메일 작성 중 검사/수정/번역하고 싶은 텍스트를 마우스로 선택

2. **확장 프로그램 실행**
   - Chrome 툴바에서 메일 서포트 아이콘 클릭
   - 선택된 텍스트가 자동으로 표시됨

3. **모드 선택**
   - **모드 1**: 맞춤법 및 문법 검사
   - **모드 2**: 메일 내용 수정 및 추천
   - **모드 3**: 한↔영 통역

4. **결과 확인 및 활용**
   - 처리된 결과 확인
   - "복사" 버튼으로 클립보드에 복사
   - 원하는 곳에 붙여넣기

## 주의사항

### API 사용 요금
- 본 확장 프로그램은 Google Gemini API를 사용합니다
- API 사용으로 인해 발생하는 요금은 **사용자 본인의 책임**입니다
- Google AI Studio에서 무료 할당량 및 요금제를 확인하세요
- 사용량을 정기적으로 모니터링하는 것을 권장합니다

### API 키 보안
- API 키는 개인 정보이므로 **절대 타인과 공유하지 마세요**
- API 키가 노출될 경우 **즉시 키를 삭제하고 새로 발급**받으세요
- API 키 노출로 인해 발생하는 모든 불이익은 **사용자 본인의 책임**입니다
- 공용 컴퓨터에서 사용 시 각별히 주의하세요

### 개인정보 보호
- API 키는 브라우저 로컬 스토리지에만 저장됩니다
- 선택한 텍스트는 Google Gemini API로 전송됩니다
- 민감한 개인정보나 기밀 정보는 신중하게 사용하세요
