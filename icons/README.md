# Icons

이 디렉토리에 다음 크기의 아이콘 이미지를 추가하세요:

- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## 아이콘 생성 방법

1. 온라인 도구 사용:
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

2. 이미지 편집 프로그램:
   - Photoshop, GIMP, Figma 등을 사용하여 직접 생성

3. 임시로 단색 PNG 파일을 생성하여 테스트할 수 있습니다.

참고: 아이콘이 없으면 확장 프로그램이 로드되지 않으므로 반드시 추가해야 합니다.
